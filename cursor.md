# How to use skills in Cursor

> 参考：https://zhuanlan.zhihu.com/p/1979612552804713312

## 1. Install openskills

```shell
npm install -g openskills
```

## 2. Install skills

```shell
openskills install <skill-name>
```

## 3. Create AGENTS.md in project root

```shell
touch AGENTS.md
```

## 4. Sync skills

```shell
openskills sync
```
